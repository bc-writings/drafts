Denote by 𝐿1(0,1)L1(0,1) the space of Lebesgue integrable functions on the interval (0,1)(0,1).

𝐐𝐮𝐞𝐬𝐭𝐢𝐨𝐧:Question: Does there exist a function 𝐹:(0,1)→ℝF:(0,1)→R such that:

  1. 𝐹(𝑥)𝑥∈𝐿1(0,1)F(x)x∈L1(0,1),
  2. 𝐹′(𝑥)𝑥∈𝐿1(0,1)F′(x)x∈L1(0,1),
  3. 𝐹(𝑥)𝑥2∉𝐿1(0,1)F(x)x2∉L1(0,1)?

I'm guessing that the answer is positive and the point is to construct 𝐹F such that 𝐹F and 𝐹′F′ behave suitably near zero. It seems quite delicate. I checked that 𝐹F cannot be a polynomial or a power function (since then 𝐹′≃𝐹𝑥F′≃Fx, thus conditions 2 and 3 cannot hold simultaneously).

I would appreciate any hints!

There is no such function. First of all, |𝐹(𝑎)−𝐹(𝑏)|⩽∫𝑏𝑎|𝐹′(𝑥)|𝑑𝑥→0|F(a)−F(b)|⩽∫ab|F′(x)|dx→0 when 𝑎,𝑏→0a,b→0. So 𝐹F has a limit 𝑐c at point 0. If 𝑐≠0c≠0, then 1) fails. So lim𝑥→0𝐹(𝑥)=0limx→0F(x)=0.

Next,

|𝐹(𝑎)|⩽∫𝑎0|𝐹′(𝑥)|𝑑𝑥⩽𝑎∫𝑎0|𝐹′(𝑥)|𝑥𝑑𝑥=𝑜(𝑎),when𝑎→0.(1)|F(a)|⩽∫0a|F′(x)|dx⩽a∫0a|F′(x)|xdx=o(a),whena→0.(1)
Now
∫𝑏𝑎𝐹(𝑥)𝑥2𝑑𝑥=𝐹(𝑎)𝑎−𝐹(𝑏)𝑏+∫𝑏𝑎𝐹′(𝑥)𝑥𝑑𝑥.(2)∫abF(x)x2dx=F(a)a−F(b)b+∫abF′(x)xdx.(2)
 Consider two cases:

  1. 𝐹F has fixed sign near 0. Then choosing 𝑎,𝑏a,b close to 0 we conclude from (1) and (2) that ∫𝐹(𝑥)𝑥2𝑑𝑥∫F(x)x2dx converges at 0, but this is equivalent to the convergence of ∫|𝐹(𝑥)|𝑥2𝑑𝑥∫|F(x)|x2dx which we need.

  2. 𝐹F has infinitely many zeroes in any neighborhood of 0. Then choosing (𝑎𝑘,𝑏𝑘)(ak,bk) being inclusion-maximal intervals of the open set {𝑥:𝐹(𝑥)≠0}{x:F(x)≠0} and applying (2) for 𝑎=𝑎𝑘,𝑏=𝑏𝑘a=ak,b=bk we bound ∫𝑐0|𝐹(𝑥)|𝑥2𝑑𝑥∫0c|F(x)|x2dx via ∫𝑐0|𝐹′(𝑥)|𝑥𝑑𝑥∫0c|F′(x)|xdx. Here 𝑐=𝑏1c=b1, for example.
