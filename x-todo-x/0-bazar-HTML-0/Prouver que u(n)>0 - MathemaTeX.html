                  Message par projetmbc » samedi 12 septembre 2020, 09:37

                  Supposons l'existence de n tel que u(n)≤0. On note alors n0 le plus petit de ces entiers. Comme u(0)>0, on sait que n0=k+1 avec k∈N. Dès lors, nous avons la contradiction u(k+1)=u(k)u(k)+1 où u(k+1)≤0 et u(k)>0.
