    By noting Ramanujan's Nested Radical, we have

    3=1+21+31+41+⋯‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√3=1+21+31+41+⋯

    On the other hand, we can manipulate the number 44 by applying the similar principle. Here we have

    4=16‾‾‾√=1+15‾‾‾‾‾‾√=1+2⋅152‾‾‾‾‾‾‾‾‾‾√=1+22254‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾√=1+21+2214‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√=1+21+3⋅22112‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√=1+21+344841144‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾⎷⋮=1+21+31+41+⋯‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√4=16=1+15=1+2⋅152=1+22254=1+21+2214=1+21+3⋅22112=1+21+344841144⋮=1+21+31+41+⋯

    How can it be? Something contradicts?

    In Ramanujan's radical, if you stop after 𝑛n nested radicals the last term inside the radical will be 1‾√1 but in your case, if you stop after 𝑛n nested radicals the last term is a term which is increasing with 𝑛n. Hence the difference.

    By Ramanujan's nested radical, we can also get this curious identity;

    𝑛+𝑚=𝑚2+𝑛𝑚2+(𝑛+𝑚)𝑚2+(𝑛+2𝑚)..‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√n+m=m2+nm2+(n+m)m2+(n+2m)..

    Ramanujan proved this identity, and if you want a look at how it's done in a simple manner (not-rigorous), check out this blog. And by plugging in 44, we get;

    3+1=1+31+41+51+6...‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√3+1=1+31+41+51+6...

    The generating function is

    𝑓(𝑥)=1+(𝑥+1)𝑓(𝑥+1)‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾√⇒𝑓2(𝑥)=1+(𝑥+1)𝑓(𝑥+1)f(x)=1+(x+1)f(x+1)⇒f2(x)=1+(x+1)f(x+1)

    so making 𝑓(𝑥)=𝑎𝑥+𝑏f(x)=ax+b we get 𝑎=1,𝑏=2a=1,b=2 or

    𝑓(𝑥)=𝑥+2f(x)=x+2
