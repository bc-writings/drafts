    Intuition behind the formula for ∑𝑖2∑i2

    Ask Question
    Asked 3 years, 5 months ago
    Active 10 months ago
    Viewed 3k times
    6

    I've been trying to figure out the intuition behind the closed formula:

    ∑𝑖2=(𝑛)(𝑛+1)(2𝑛+1)6∑i2=(n)(n+1)(2n+1)6

    This is not hard to prove via induction, so I'm not interested in the proof that this is true. Rather, I'm more interested in why somebody would even make this guess for an inductive step. What's so confusing to me is that

    ∑𝑖=(𝑛)(𝑛+1)2∑i=(n)(n+1)2 makes sense using the Gaussian trick of taking the number line and folding it in half and realizing that (1+n) = (2+(n-1)) = (3 +(n-2))... so that you end up with this form.

    But, what's the intuition for 𝑖2i2? Looking at these two formulas, one realizes very quickly that

    ∑𝑖2=∑𝑖∗(2𝑛+2)3∑i2=∑i∗(2n+2)3

    But, why is that true intuitively? What's the intuition for this?

    Since you asked for an intuitive explanation consider a simple case of 12+22+34+4212+22+34+42 using a set of children's blocks to build a pyramid-like structure.

    First you arrange 1616 blocks in a 4×44×4 square. Next you place on top of this arrangement, 99 blocks in a 3×33×3 square aligning the blocks of the upper left corner of each square, one above the other. On top of this construction place 44 blocks in a 2×22×2 square, similarly aligned. Finally crown the upper left corner with a single block for the 1×11×1 square.

    The following table represents the number of block in each column of the arrangement viewed from above:

    43213321222111114321332122211111

    We can find the total number of blocks in the arrangement by adding the number of columns containing a single block to twice the number of columns containing two blocks then adding three times the number of columns containing three blocks and finally adding four times the one column containing four blocks.

    ∑𝑖=14𝑖2=1⋅(4+3)+2⋅(3+2)+3⋅(2+1)+4⋅1∑i=14i2=1⋅(4+3)+2⋅(3+2)+3⋅(2+1)+4⋅1

    If we do the same thing with a pyramid of blocks having 𝑛×𝑛n×n blocks on its base then the summation would look like the following:

    ∑𝑖=1𝑛𝑖2∑𝑖=1𝑛𝑖23∑𝑖=1𝑛𝑖2∑𝑖=1𝑛𝑖2=+=======1⋅(𝑛+𝑛−1)+2⋅(𝑛−1+𝑛−2)+3⋅(𝑛−2+𝑛−3)⋯+(𝑛−1)⋅(2+1)+𝑛⋅11⋅(2𝑛−1)+2⋅(2𝑛−3)+3⋅(2𝑛−5)+⋯+(𝑛−1)⋅3+𝑛⋅1∑𝑖=1𝑛𝑖(2𝑛−2𝑖+1)2𝑛∑𝑖=1𝑛𝑖−2∑𝑖=1𝑛𝑖2+∑𝑖=1𝑛𝑖(2𝑛+1)∑𝑖=1𝑛𝑖−2∑𝑖=1𝑛𝑖2(2𝑛+1)∑𝑖=1𝑛𝑖𝑛(𝑛+1)(2𝑛+1)2𝑛(𝑛+1)(2𝑛+1)6∑i=1ni2=1⋅(n+n−1)+2⋅(n−1+n−2)+3⋅(n−2+n−3)+⋯+(n−1)⋅(2+1)+n⋅1=1⋅(2n−1)+2⋅(2n−3)+3⋅(2n−5)+⋯+(n−1)⋅3+n⋅1=∑i=1ni(2n−2i+1)=2n∑i=1ni−2∑i=1ni2+∑i=1ni∑i=1ni2=(2n+1)∑i=1ni−2∑i=1ni23∑i=1ni2=(2n+1)∑i=1ni=n(n+1)(2n+1)2∑i=1ni2=n(n+1)(2n+1)6

    There is this really cool thing called Pascal's triangle. Perhaps you've heard of it, and it is shown below;

    You may also know of the Hockey Stick Identity:

    Amazingly, the Hockey Stick Identity can be used to almost directly show that

    ∑𝑘=1𝑛𝑘(𝑘+1)(𝑘+2)…(𝑘+𝑝)=𝑛(𝑛+1)(𝑛+2)…(𝑛+𝑝+1)𝑝+2∑k=1nk(k+1)(k+2)…(k+p)=n(n+1)(n+2)…(n+p+1)p+2

    See then that if 𝑝=1,0p=1,0, we have

    ∑𝑘=1𝑛𝑘2=∑𝑘=1𝑛𝑘(𝑘+1)−𝑘=𝑛(𝑛+1)(𝑛+2)3−𝑛(𝑛+1)2=𝑛(𝑛+1)(2𝑛+1)6∑k=1nk2=∑k=1nk(k+1)−k=n(n+1)(n+2)3−n(n+1)2=n(n+1)(2n+1)6

    I like to use this because the general sum involving 𝑝p can be derived as a combination sum by dividing both sides by (𝑝+1)!(p+1)!. It also happens to be easy to remember and generally works well for deriving formulas for ∑𝑘𝑎∑ka. Just to show how it would work for 𝑘3k3,

    ∑𝑘=1𝑛𝑘3=∑𝑘=1𝑛𝑘(𝑘+1)(𝑘+2)−3𝑘(𝑘+1)+𝑘=𝑛(𝑛+1)(𝑛+2)(𝑛+3)4−3𝑛(𝑛+1)(𝑛+2)3+𝑛(𝑛+1)2=𝑛2(𝑛+1)222∑k=1nk3=∑k=1nk(k+1)(k+2)−3k(k+1)+k=n(n+1)(n+2)(n+3)4−3n(n+1)(n+2)3+n(n+1)2=n2(n+1)222

    Taking differences once give 𝑖2i2, again gives 2𝑖−12i−1, and a third time gives a constant 22.

    This suggests the formula should be a third-degree polynomial with leading coefficient 23!=1323!=13 as you might expect if you integrate 22 three times.

    It is then not difficult to find the solution: one way is to look at ∑𝑖2−𝑛33∑i2−n33 and take its first and second difference to get a constant. In the end this will give an inductive hypothesis of

    ∑𝑖=1𝑛𝑖2=𝑛(𝑛+1)(2𝑛+1)6∑i=1ni2=n(n+1)(2n+1)6
    which is a typo away from your question
